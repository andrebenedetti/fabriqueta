# Decision Log

This directory contains architectural decision records (ADRs) for the Fabriqueta project.
Each decision record follows the template in `_templates/decision-record.md`.

## What to Record

Record any significant decision that affects the product, architecture, or operating model:
- Technology choices
- Architecture changes
- Process changes
- Product scope decisions
